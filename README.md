# Module 3 - Инвертированный индекс и сжатие

## Запуск тестов
python test_index.py

## Структура проекта
- compression.py   # реализация сжатия (Элиас, Голомб)
- inverted_index.py # инвертированный индекс
- main.py          # общий запуск
- test_index.py    # тестирование и эксперименты
- data/            # папка с данными (pages.jsonl)

## Необходимые данные
Файл pages.jsonl должен быть помещён в папку data/, каждая строка в формате: {"url": "...", "text": "..."}
